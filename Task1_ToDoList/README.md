# Task 1 - To-Do List Application

## Description

BeginNow is a GUI-based To-Do List application developed using Python and Tkinter. It helps users organize daily tasks efficiently.

## Features

* Add Tasks
* Edit Tasks
* Delete Tasks
* Mark Tasks as Completed
* Clear All Tasks
* Persistent Storage using JSON

## Technologies Used

* Python
* Tkinter
* JSON

## How to Run

```bash
python main.py
```
