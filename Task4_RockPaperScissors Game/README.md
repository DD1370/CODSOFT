# 🪨 Rock Paper Scissors Game

A simple and interactive **Rock Paper Scissors Game** built using **Python Tkinter**. The application allows users to play against the computer, track scores, view game history, and determine the overall winner.

---

## 📌 Features

* 🎮 User-friendly graphical interface using Tkinter
* 🪨 Rock, 📃 Paper, and ✂️ Scissors selection buttons
* 🤖 Random computer choice generation
* 🏆 Automatic winner determination
* 📊 Real-time score tracking
* 🔄 Reset Game functionality
* 📜 Game history tracking
* 📢 Status updates for each round
* 🚪 Exit confirmation dialog
* 🏅 Final winner announcement when exiting
* 🎨 Clean and attractive blue-themed interface

---

## 🛠 Technologies Used

* Python 3
* Tkinter (GUI Library)
* Random Module

---

## 🎯 Game Rules

* 🪨 Rock beats Scissors
* ✂️ Scissors beats Paper
* 📃 Paper beats Rock
* Same choices result in a Tie
