# 🔐 Password Generator (Python)

A simple yet powerful **Password Generator application** built using Python.
It generates strong and secure passwords based on user-defined length and complexity options such as numbers and special characters.

---

## 🚀 Features

* Generate secure random passwords
* User-defined password length
* Option to include:

  * 🔢 Numbers
  * 🔣 Special characters
* Input validation (handles invalid entries safely)
* Strong password generation ensuring required criteria
* Beginner-friendly command-line interface

---

## 🛠️ Technologies Used

* Python 3
* `random` module
* `string` module

---
## 🧩 How It Works

1. User enters desired password length
2. User selects:

   * Include numbers (yes/no)
   * Include special characters (yes/no)
3. Program:

   * Builds a secure character pool
   * Ensures required character types are included
   * Generates a randomized password
4. Final password is displayed

---
